# News Spot

Прототип новостного сайта с поиском, просмотром и избранным.

## О проекте

работа по практике, по заданию.

## Как скачать проект

1. Склонируйте репозиторий:

```bash
git clone https://github.com/Davedus877/github
cd gut hub
```


2. Установите зависимости:

```bash
npm install
```

## Как запустить проект

1. Скопируйте файл `.env.example` в `.env`:

```bash
copy .env.example .env
```

2. В файле `.env` укажите свой ключ `NEWS_API_KEY` от NewsAPI.

3. Запустите сервер:

```bash
npm run dev
```

4. Откройте в браузере:

```text
http://localhost:6969
или 
http://localhost:6767
```

## Функции

- Поиск новостей
- Просмотр результатов
- Сохранение новостей в избранное
- Хранение избранного в браузерном `localStorage`
